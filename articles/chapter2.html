给文档添加样式的三种方式

  • 行内样式: 行内样式只能影响它所在的标签,而且总会覆盖嵌入样式和链接样式。 

    
    <p style="font-size: 12px; font-weight:bold; font-style:italic; color:red;"></p>

  • 嵌入样式: 嵌入的CSS样式是放在HTML文档的head元素中的。嵌入样式的应用范围仅限于当前页面,嵌入样式会覆盖外部样式表(浏览器默认样式表,用户样式表,链接样式表)中的样式,但会被行内样式所覆盖。 

    
    <head>
        <style type="text/css">
            h1 {font-size:16px;}
            p {color:blue;}
        </style>
    </head>

  • 链接样式: 在HTML文档页面从外部引入一个扩展名为.css的文本文件, 这个文件就叫样式表。 

    
    <link href="styles.css" rel="stylesheet" type="text/css" />

CSS的基本语法

  • 为一个选择符设置多个样式: 

    
    p { color:red; font-size:12px; font-weight:bold; }

  • 多个选择符设置相同的样式: 

    
    h1 { color:blue; font-weight:bold; }
    h2 { color:blue; font-weight:bold; }
    h3 { color:blue; font-weight:bold; }

    h1, h2, h3 { color:blue; font-weight:bold; }

  • 多条规则应用到一个选择符: 

    
    h1, h2, h3 { color:blue; font-weight:bold; }
    h3 { font-style:italic; }
