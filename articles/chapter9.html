假如你在大街上遇到一个黑人, 请猜猜这人很可能来自哪里?
(A: 亚洲, B: 非洲, C: 欧洲)

事件H: 你在大街上遇到一个黑人, 记事件H发生的概率为P(H)

事件A: 这人来自亚洲, 记事件A发生的概率为P(A)

事件B: 这人来自非洲, 记事件B发生的概率为P(B)

事件C: 这人来自欧洲, 记事件C发生的概率为P(C)

P(AH) = ?

$P\left( AH \right) \quad =\quad P(A)\cdot P(H|A)\quad =\quad P(H)\cdot P(A|H)\\ P(A|H)\quad =\quad \frac { P(A)\cdot P(H|A) }{ P(H) } \\ P(B|H)\quad =\quad \frac { P(B)\cdot P(H|B) }{ P(H) } $
